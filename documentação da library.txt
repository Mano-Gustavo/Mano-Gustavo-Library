CARREGAMENTO REMOTO VIA GITHUB (RECOMENDADO)

A library é carregada diretamente do GitHub, evitando scripts gigantes e facilitando atualizações.

Use SEMPRE este código no início do seu script:

local Library = loadstring(game:HttpGet(
"https://raw.githubusercontent.com/Mano-Gustavo/Mano-Gustavo-Library/refs/heads/main/library.lua"
))()

==================================================

[1] CRIANDO A JANELA PRINCIPAL

A janela é a base do seu Hub.

Exemplo:

local Window = Library:CreateWindow({
Title = "Meu Hub",
Author = "Seu Nome",
Keybind = Enum.KeyCode.RightControl
})

Explicação:

Title: Nome do hub (aparece no topo)

Author: Nome do criador (opcional)

Keybind: Tecla para abrir/fechar no PC

OBS:
No celular, um botão flutuante e arrastável aparece automaticamente.

==================================================

[2] SISTEMA DE ABAS (TABS)

As abas servem para organizar as funções do seu script.

Exemplo:

local TabMain = Window:CreateTab("Principal")
local TabESP = Window:CreateTab("ESP")

Tudo que for criado usando a variável da aba aparecerá dentro dela.

==================================================

[3] ELEMENTOS DISPONÍVEIS NA LIBRARY

A) BUTTON — Botão simples
Executa uma ação uma única vez ao clicar.

Tab:CreateButton("Executar", function()
print("Botão clicado")
end)

B) TOGGLE — Liga / Desliga
Usado para sistemas contínuos (Auto Farm, ESP, etc).

Tab:CreateToggle("Auto Farm", function(Value)
getgenv().Farm = Value
while getgenv().Farm do
task.wait(0.1)
end
end, false)

C) SLIDER — Barra numérica
Usado para definir valores entre mínimo e máximo.

Tab:CreateSlider("Velocidade", 16, 200, 16, function(Value)
game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = Value
end)

D) DROPDOWN — Lista de opções

Tab:CreateDropdown("Selecionar", {"Opção 1", "Opção 2"}, function(Value)
print(Value)
end)

E) TEXTBOX — Entrada de texto

Tab:CreateTextBox("Nome", "Digite aqui...", function(Text)
print(Text)
end)

F) LABEL — Texto informativo
Usado apenas para organização visual.

Tab:CreateLabel("--- Configurações ---")

G) COLOR PICKER RGB (NOVO)

Permite escolher cores personalizadas usando RGB.
Ideal para ESP, Chams, Wallhack e UI.

Tab:CreateColorPicker("Cor do ESP", Color3.fromRGB(255, 0, 0), function(Color)
print(Color)
end)

Informações importantes:

Retorna um Color3

Atualiza em tempo real

Pode ser aplicado diretamente em Highlight, UI, etc.

[4] SISTEMA DE NOTIFICAÇÕES AVANÇADO

A library possui notificações por tipo, facilitando feedback visual.

Uso básico:

Library:Notification({
Title = "Título",
Text = "Mensagem",
Duration = 5,
Type = "Info"
})

Tipos disponíveis:

Success → Verde (sucesso)

Warning → Amarelo (aviso)

Error → Vermelho (erro)

Info → Azul (informação)

Exemplo:

Library:Notification({
Title = "Sucesso!",
Text = "Auto Farm ativado.",
Duration = 4,
Type = "Success"
})

[5] SISTEMA DE LUPA / PESQUISA (NOVO)

A library possui um sistema de pesquisa (lupa) integrado.

Função:

Permite pesquisar elementos dentro da aba atual

Filtra em tempo real

Elementos filtrados:

Buttons

Toggles

Sliders

Dropdowns

TextBox

Color Picker

Labels

Observações:

Atua apenas na aba aberta

Não afeta performance

Ideal para hubs grandes
